# pygbif
- GBIFデータをPythonで扱うためのライブラリ
- https://pygbif.readthedocs.io/en/latest/index.html
- https://github.com/gbif/pygbif

## インストール
### pip
- オフィシャルサイトにはこれが載っている
```
pip install pygbif
```
- Develop版の入れ方は本家サイト参照
### conda
```
conda install conda-forge::pygbif
```
- 本家conda（main）には入っていなくて、ボランティアがアップしているconda-forgeの方に入っている
